Lista Ordenada

Para listas ordenados usaremos a tag ol

e para identificar cada item a tag li

Exemplo:
    Rotina
  1. acordar
  2. tomar banho
  3. ir para o estágio
  4. treinar
  5. estudar
Caso queira mudar o estilo da lista utiliza-se
type= "i", "I", "A", "a", após a tag.
O número 1 já vem por padrão. Exemplo:
  1. puxada alta
  2. remada curvada
  3. remada cavalinho

Lista não ordenada

Para listas não ordenadas usamos a tag ul

Exemplo:
    Comida
  • omelete
  • pão
  • vitamina
  • café

    • Para mudar o estilo dos pontos, usaremos type= "circle, "square","disc".

    Exemplo:
    • ssd
    • ram
    • processador
    • placa de vídeo

    Listas de definição

    As tagas são dl, dt para termo e dd para descrição.

    Exemplo:
    Definição de sentimentos
    Amor
    Forte afeição por outra pessoa, nascida de laços de consanguinidade ou de relações sociais.
